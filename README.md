# 🚀 SoftITo Backend Engineering Journey: 320 Hours to Mastery

Welcome to my digital diary and master portfolio showcasing my transition into a Backend Engineer. This repository serves as the central hub and navigation dashboard for my **320-hour intensive Backend Engineering Training Program** organized by **SoftITo** in collaboration with the **Istanbul Chamber of Commerce (İTO)**.

---

## 🛠️ Tech Stack & Skills Acquired
Across both phases of this intensive journey, I have developed expertise in the following technologies and architectural concepts:

- **Core & OOP:** C#, .NET 8 / .NET 10, Object-Oriented Programming (OOP)
- **Database & Querying:** MS SQL Server, T-SQL, ADO.NET, Dapper, Entity Framework Core (Code-First & Database-First), LINQ
- **Web Development:** ASP.NET Core MVC, Razor Pages, RESTful Web APIs, Swagger (OpenAPI)
- **Architectural Patterns:** N-Tier (Layered) Architecture, Repository Pattern, Decoupled Systems (API + Client), Data Transfer Objects (DTOs)
- **Optimization & Tooling:** In-Memory Caching, Rate Limiting (Sliding Window), Reporting (QuestPDF, ClosedXML), Auditing, MS Identity Authentication, Git & GitHub

## 📈 Key Learnings & Growth
1. **Architectural Thinking:** Moved from foundational desktop/console apps to decoupled, multi-tier enterprise web architectures and RESTful APIs.
2. **Database Management:** Gained hands-on experience utilizing ORM technologies (like Entity Framework Core or Dapper) alongside raw SQL queries for optimal data storage and access.
3. **API Design & Standards:** Developed clean, structured APIs emphasizing HTTP standard verbs, status codes, DTO mapping, and Swagger documentation.
4. **Performance & Security:** Explored caching, rate limiting, and secure user authentication flows.
